# Module 5: CI/CD for Test Engineers – Assignments & Exploration

## Assignment 1: CI Integration
- Create the test repo (Python or JS) for assignments of all modules
- Add a GitHub Actions workflow to run your tests automatically
- Generate and upload Allure report as an artifact

---

## Assignment 2: Dockerize Your Tests
- Write a Dockerfile for your automation suite
- Run it locally and in CI

---

## Assignment 3: Observability Challenge
- Export your test metrics (pass rate, response times) to Grafana Cloud or ELK
- Create one dashboard panel visualizing test health
